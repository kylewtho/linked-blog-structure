# How to Add Content

This tutorial shows how to add a new post to your digital garden.

## 1. Create a Markdown File

Create a new `.md` file in the `common_md/` directory or its subfolders.

## 2. Add Frontmatter

Start your file with a YAML frontmatter block:

```yaml
---
title: My New Post
excerpt: A brief summary.
date: "2026-04-17"
tags: [news, update]
---
```

## 3. Write Your Content

Use standard Markdown. You can link to other posts using `[Title](file.md)` or `[[file]]`.

## 4. Add Images

Place images in `common_md/attachments/` and embed them with `![[image-name.png]]`.
