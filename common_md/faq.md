# Frequently Asked Questions

## General Questions

### What is a "Digital Garden"?
A digital garden is a collection of evolving notes and ideas, rather than just chronological blog posts. It emphasizes [[linking-and-backlinks]] between topics.

### How is this site built?
This site is built using **Next.js**, **Markdown**, and **Tailwind CSS**. It uses a custom engine to process Markdown files in the `common_md` directory.

## Technical Questions

### How do I add a new post?
Follow the [Sample Tutorial](tutorials/sample-tutorial.md) for a step-by-step guide.

### Does it support syntax highlighting?
Yes! See the [Markdown Features Demo](markdown-features.md) for examples of code blocks.

### How do I link to another post?
You can use standard Markdown links `[Title](file.md)` or wikilinks `[[file]]`.
